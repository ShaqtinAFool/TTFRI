## 通用版

恭喜您通過審核！  
  
再次提醒，  
本資料庫目前僅供學術研究用途或政府單位使用，  
請勿將資料用於商業用途或提供予第三方使用。  
(詳見會員申請同意書第三條第一項紅字部分[1])  
  
若您所產出的成果有意願致謝本資料庫，  
請參考成果回饋頁面所提供的資訊[2]，  
以利颱洪中心依此爭取更多服務資源。

請見資料總表[3]。(首頁=>關於我們=>資料總表)

或於滿意度調查中提供您的建議與想法，  
相關內容將作為本資料庫未來擴充的參考依據，  
謝謝您的配合。  
  
預祝研究順利!
